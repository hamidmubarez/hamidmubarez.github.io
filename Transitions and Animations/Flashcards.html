Basic Web Design Flashcards Review

HTML

HTML is the acronym, or abbreviation, for HyperText Markup Language and gives a website its structure

HyperText

HyperText refers to the ability to navigate between webpages using hyperlinks

Hyperlinks

Hyperlinks are the images or text that you click to go to another website.

Markup Language

Markup Language is text that can be interpreted by a browser to determine a webpage's format, style, and structure through the use of markup tags

Elements

Those items that result when the HTML has been parsed are known as elements,Elements help establish a website's readability and visual hierarchy

Attributes

The purpose of an attribute is to modify the HTML element it accompanies

CSS

CSS stands for Cascading Style Sheets. It is another markup language, except its purpose is presentation

JavaScript

JavaScript is a versatile scripting language that is commonly used in conjunction with HTML and CSS to create dynamic and interactive web pages
